Deverá ser criada uma PWA utilizando o framework Angular.

Essa PWA deverá conter:

- Uma listagem de usuários; ok
- Inclusão de um novo usuário; ok
- Edição dos dados do usuário; ok
- Possibilidade de exclusão do usuário;
- Rota de 404;

Os dados obrigatórios do usuário são: Nome completo, CPF, e-mail e endereço (CEP, Rua ou Avenida, número e complemento). OK
Só podemos acessar a listagem e todas as outras funcionalidades depois que for realizado o login, ou seja, é necessário criar uma tela de autenticação.

Outros pontos obrigatórios:
- Precisam ser criados dois @Pipe do Angular, são eles: uma para filtrar e outro para ordenação;
- Usar as coleções do Firebase;
- Precisa estar no Hosting do Firebase;

Pontos que serão levados na avaliação:
- Organização do código;
- Facilidade de leitura;
- Interface (pense no seu usuário);
- Readme.md precisa ser com as suas palavras, caso contrário será descontado 1 ponto;
  -- Neste Readme.md eu preciso ser capaz de executar o projeto e entender como realizar o login.
- Versionamento da pasta node_modules será descontado 1 ponto;
- Imagens padrão que vem no módulo de PWA do Angular precisam ser modificadas, caso contrário, o desconto será de 0,5 ponto;
